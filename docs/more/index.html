Attribution

The design of this site is based on Tom MacWright’s wonderful, minimal site. The main typeface is Inter, and the monospace font is JetBrains Mono. The serifs are Playfair Display. I wanted to stick with system fonts, but I really like the look of the typefaces.

I try to use my own drawings and media files whenever I can, but if not, it’s most likely sourced from unsplash, pixabay or pexels. The entire site is built using Zola.
