Spotify Wrapped

This is one of the more experimental posts.

Spotify annually releases Spotify Wrapped — a visual compilation of user listening data for the calendar year (roughly). My biggest gripe with it is that it does not show data from previous years. I like being able to see how my music tastes have evolved over the years rather than just the current year.

So I manually downloaded all my data, and had a go at visualising it myself using D3. The source is available here.

Personally, I think the calendar plot is more data dense and gives deeper insights than listing top artists and top songs of the year. But there’s also disadvantages — the most obvious being that it will not scale well at all for smaller screens.
